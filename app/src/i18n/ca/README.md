# Guia de Terminologia

Fem servir una terminologia consistent a tota l'aplicació per garantir claredat i alineació amb els nostres valors. Les nostres tries de paraules busquen:

1.  **Ser intuïtives** - Fàcils d'entendre per als usuaris
2.  **Evitar argot financer** - No utilitzar "diners", "pagament", "pagar", "pagador", "transacció"
3.  **Utilitzar la metàfora d'enviar *tokens*** - Emmarcar les interaccions com a enviar/rebre, però sense utilitzar la paraula "token"

## Termes Principals

| Terme | Definició | Notes d'Ús |
| :--- | :--- | :--- |
| **Comunitat** | La xarxa local d'intercanvi dins de l'aplicació | Utilitzar en lloc de "moneda" quan es refereix al grup en si. Abans era "grup" o "xarxa" |
| **Moneda** | El *token* utilitzat per una comunitat | Utilitzar només quan es refereix al token explícitament i no al conjunt de persones |
| **Membre** | Un usuari/participant a l'aplicació | Té nom, perfil, avatar. Utilitzar en lloc de "compte" quan es refereix a la persona |
| **Compte** | La cartera d'un membre | Conté el saldo i l'historial de moviments |
| **Saldo** | Quantitat actual de moneda en un compte | |
| **Moviment** | Moviment de moneda entre comptes | Abans era "transacció" |
| **Enviar** | Iniciar un moviment des del meu compte | |
| **Rebre** | Iniciar un moviment cap al meu compte | |
| **Moure** | Iniciar un moviment entre altres comptes | Acció només per a administradors |
| **De / Origen** | Origen d'un moviment | Utilitzar "De" en etiquetes, "origen" en text més llarg. Abans era "pagador" |
| **A / Destinació** | Destinació d'un moviment | Utilitzar "A" en etiquetes, "destinació" en text més llarg. Abans era "cobrador" |
| **Quantitat** | Quantitat de moneda en un moviment | |
| **Oferta** | Un bé o servei que un membre ofereix | |
| **Necessitat** | Un bé o servei que un membre cerca | |
| **Valor** | Preu d'una oferta o necessitat | Abans era "preu" |
