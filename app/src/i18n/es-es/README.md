# Guía de Terminología

Usamos terminología consistente en toda la aplicación para garantizar claridad y alineación con nuestros valores. Nuestras elecciones de palabras buscan:

1.  **Ser intuitivas** - Fáciles de entender para los usuarios
2.  **Evitar jerga financiera** - No usar "dinero", "pago", "pagar", "pagador", "transacción"
3.  **Usar la metáfora de "enviar *tokens*"** - Enmarcar las interacciones como enviar/recibir, pero sin usar la palabra "token"

## Términos Principales

| Término | Definición | Notas de Uso |
| :--- | :--- | :--- |
| **Comunidad** | La red local de intercambio dentro de la aplicación | Usar en lugar de "moneda" cuando se refiere al grupo en sí. Antes era "grupo" |
| **Moneda** | La ficha usada por una comunidad | Usar solo cuando se refiere a la ficha explícitamente |
| **Miembro** | Un usuario/participante en la aplicación | Tiene nombre, perfil, avatar. Usar en lugar de "cuenta" cuando se refiere a la persona |
| **Cuenta** | La cartera de un miembro | Contiene el saldo y el historial de movimientos |
| **Saldo** | Cantidad actual de moneda en una cuenta | |
| **Movimiento** | Movimiento de moneda entre cuentas | Antes era "transacción" |
| **Enviar** | Iniciar un movimiento desde mi cuenta | |
| **Recibir** | Iniciar un movimiento hacia mi cuenta | |
| **Mover** | Iniciar un movimiento entre otras cuentas | Acción solo para administradores |
| **De / Origen** | Origen de un movimiento | Usar "De" en etiquetas, "origen" en texto más largo. Antes era "pagador" |
| **A / Destino** | Destino de un movimiento | Usar "A" en etiquetas, "destino" en texto más largo. Antes era "beneficiario" |
| **Cantidad** | Cantidad de moneda en un movimiento | |
| **Oferta** | Un bien o servicio que un miembro ofrece | |
| **Necesidad** | Un bien o servicio que un miembro busca | |
| **Valor** | Precio de una oferta o necesidad | Antes era "precio" |
